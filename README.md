# 📌 Rick and Morty App

> Application to list all characters, locations, and episodes of Rick and Morty and their details.

## 🚀 Tech Stacks

- **Retrofit** - HTTP requests  
- **Jetpack Compose** - UI toolkit  
- **Coil** - Image renderer  
- **Hilt** - Dependency injection  

## 🏗️ Architecture  

- **MVVM (Model-View-ViewModel)**
